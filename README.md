# Exment ガントチャートプラグイン
Exment用のガントチャートビューです。

公式プラグインではありません。

色は赤、青、緑に対応(選択列等で設定)
※必要な色があればcssに追加してPlugin.phpファイルにも色部分も追加してください。

| 列の名前 | 列種類 |
| --- | --- |
| タイトル | 1行テキスト等 |
| 開始日 | 日付(日付と時刻含む) |
| 終了日 | 日付(日付と時刻含む) | 
| 進捗度 | 整数(0~100) |
| 色 | 赤、青、緑(初期値　追加はCSSから) またはカラーコードを使用|

意図しないドラッグを防ぐため編集モードは更新時等はオフなっています。切り替えて使用してください。

# 土日の追加ハイライト方法
gantt.blade.phpのstyleでいいので以下を追加
