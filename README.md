# Exment ガントチャートプラグイン

| 列の名前 | 列種類 |
| --- | --- |
| タイトル | 1行テキスト等 |
| 開始日 | 日付(日付と時刻含む) |
| 終了日 | 日付(日付と時刻含む) | 
| 進捗度 | 整数(0~100),選択列(テキストマッピングが必須) |
| 分類 | 分類(カテゴリー分け)できるように追加 |
| 色 | 赤、青、緑など またはカラーコードを使用|
