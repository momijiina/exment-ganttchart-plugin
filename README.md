# Exment ガントチャートプラグイン
Frappe Ganttを廃止しシンプルなガントチャートへ変更します。<br/>
旧バージョンとの互換性がありません。<br/>
大量データが存在する場合？にフィルタをかけても一部、データが表示されない不具合あり。

公式プラグインではありません。

色は赤、青、緑に対応(選択列等で設定)
または、カラーコード

| 列の名前 | 列種類 |
| --- | --- |
| タイトル | 1行テキスト等 |
| 開始日 | 日付(日付と時刻含む) |
| 終了日 | 日付(日付と時刻含む) | 
| 進捗度 | 整数(0~100) |
| 色 | 赤、青、緑(初期値　追加はCSSから) またはカラーコードを使用|
